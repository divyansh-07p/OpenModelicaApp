# OpenModelica Desktop Simulation App

##  Overview
This project is a desktop application built using PyQt that allows users to execute OpenModelica simulation models through a simple and intuitive graphical interface.

Instead of running simulations manually via terminal, this tool provides a user-friendly way to launch compiled models using a `.bat` executable generated by OpenModelica.

---

##  Objectives
- Simplify execution of OpenModelica models
- Provide a GUI-based alternative to command-line simulation
- Ensure reliable execution across systems

---

##  Features
-  File selection dialog for `.bat` executable
-  Input fields for simulation start and stop time
-  One-click simulation execution
-  Input validation (numeric + constraints)
-  Error handling with user-friendly messages
-  Cross-system compatible approach using `.bat`

---

cd OpenModelicaApp


2. Install dependencies:

pip install -r requirements.txt


3. Ensure OpenModelica is installed on your system.

##  How to Run


python main.py


---

##  Usage Guide

1. Click **Browse** and select the `.bat` file generated by OpenModelica.
2. Enter:
   - Start Time (e.g., 0)
   - Stop Time (e.g., 1)
3. Click **Run Simulation**
4. View success or error messages via GUI

---

##  Constraints
- `0 ≤ start time < stop time < 5`
- Inputs must be numeric

---

##  Design Decisions

### Why `.bat` instead of `.exe`?
While OpenModelica generates both `.exe` and `.bat` files, this application uses the `.bat` file for execution.

**Reason:**
- `.bat` ensures correct runtime environment setup
- `.exe` execution may fail on systems without proper configuration (DLL/path issues)

This design choice improves **reliability and portability** of the application.
